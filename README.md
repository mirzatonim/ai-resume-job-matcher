# AI Resume–Job Matcher

AI-powered system that analyzes how well a resume matches a job description using real NLP metrics.

## Tech Stack
- Spring Boot (Java)
- FastAPI (Python)
- SentenceTransformers
- TF-IDF
- NLP similarity scoring

## Architecture
Spring Boot → AI NLP Service → ML Models

## Features
- Resume–Job similarity score
- Skill match score
- Experience match score
- Keyword relevance
